
A weather widget for the terminal ⛅️

![image](https://github.com/pawap90/terminal-weather-widget/assets/2507959/d49d8a7b-a762-47e7-9e8c-8e21ef50eb57)

## Setup

```
npm install
```

## Run it

```
npm run build
npm start
```

## Development

```
npm run dev
```

## Run as CLI

```
npm run build

npm link

weather --lat -34.6037 --lon -58.3816 --timezone GMT
```
