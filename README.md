
A weather widget for the terminal ⛅️

![image](https://github.com/pawap90/terminal-weather-widget/assets/2507959/9d1166dc-d8ef-4b89-97d6-240ec86e365b)

## Setup

```
npm install
```

## Run it

```
npm run build
npm start
```

## Development

```
npm run dev
```

## Run as CLI

```
npm run build

npm link

weather --lat -34.6037 --lon -58.3816 --timezone GMT
```
